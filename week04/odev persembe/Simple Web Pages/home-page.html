Web Basics

It is the first module of our course. This course first covers the basics of web technologies. The tools required for programming websites are presented. The technologies listed below will be completed in approximately 8 weeks with an intense homework pace. Participants who have completed this course will have gained knowledge end experience in the use of following tools and technologies and with this knowledge and experience will be able to create static websites very easily.

Web Programming

In this second course of our module, basic programming structures in Javascript are taught. The tasks dealt with in this module are well above the basic level, and the submission of these tasks is the most important prerequisite for completing course. While the participants prepare these assigments, they need to do search outside of class and, in particular, have frequent dialogues with their mentors and request their support. Although this module is a very challenging module, it is main module that teaches programming beyond the basics.
